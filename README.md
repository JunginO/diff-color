# [NUMBLE] 다른 색깔 찾기 게임 제작 챌린지
- 넘블에서 진행된 챌린지로, Context, Redux, Recoil 등 상태관리 라이브러리를 사용하지 않고 다른 색깔 찾기 게임을 제작하는것이 목적
- 요구사항에 따라 적절한 단위로 component 나누기, component의 state 등을 연습

### 사용한 라이브러리
styled component를 이용해 박스 색상과 넓이를 관리했다
